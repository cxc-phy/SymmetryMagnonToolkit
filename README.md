# Project Overview

This repository contains source code and example data for the current project.

## Structure

- `symmetry-analyzer/`: symmetry analysis related code and databases.
- `magnon-solver/`: magnon solver source code.
- `demo/`: sample inputs and outputs prepared for other people to test the project quickly.
- `results/`: generated reference results and documents.

## Demo

The `demo/` directory is intentionally tracked in Git so collaborators and testers can use it directly as a runnable example set.

## Git Notes

This repository ignores temporary notebook checkpoints and build artifacts such as `*.o`, `*.mod`, `*.exe`, and `magnon-solver/compile/`.
